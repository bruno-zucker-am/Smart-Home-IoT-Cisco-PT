# Smart-Home-IoT-Cisco-PT
Simulação de uma rede residencial inteligente com telefonia IP (Cisco SCCP), dispositivos IoT gerenciados via Home Gateway isolado, DNS externo customizado (ns.casa.com.br) e DHCP configurado no roteador Cisco 2901. Switch Catalyst 2960 com portfast e voice VLAN. Desenvolvida no Cisco Packet Tracer como parte de um portfólio prático de redes.
